## Commands
- ### ***!play***<br>
     *Play a video or a playlist on YouTube*

- ### ***!skip***<br>
     *Skip a song*

- ### ***!pause***<br>
     *Pause the song*

- ### ***!resume***<br>
     *Resume the song*

- ### ***!show***<br>
     *Show the next ten songs*

- ### ***!exit***<br>
     *Exit voice channel*

- ### ***!help***<br>
     *Show the help message*
